# 10 · Ecosystem

LiteTV OS is the anchor of a broader cross-device ecosystem.

## The Lite Ecosystem

```
LiteTV OS → LitePhone OS → LitePad OS → LiteWatch OS
          → LiteCloud → LiteAI → LiteHome
          → Developer SDK → Theme Store → Plugin System
```

## Branding

| Element | Value |
|---------|-------|
| Name | LiteTV OS |

## LiteConnect — Cross-Device

- Phone ↔ TV pairing
- Clipboard Sync
- File Sharing
- Media Handoff
- Remote Input
- Notification Sync

## LiteCloud

Backs up: Settings · Installed Apps · Favorites · Wallpaper · User Profiles ·
OTA History. See [OTA](09-OTA.md).

## LiteRemote

Companion app for Android and iOS:

- Touchpad
- Keyboard
- Voice Input
- Media Controls
- Power
- Input Switching
- Volume

## LiteCast — Wireless Display

- Screen Mirroring
- Media Streaming
- Photo Sharing
- Music Streaming
- Low Latency Mode

## LiteAI — Offline AI

- Voice Commands
- Search
- Settings Control
- App Launcher
- Media Search
- Device Diagnostics
- *Future:* Local LLM

See [AI Management](16-AI-Management.md) for the on-device architecture, resource
model, and privacy controls.
